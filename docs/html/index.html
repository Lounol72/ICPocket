Présentation du Projet

ICPocket est un jeu de combat et de capture de créatures, développé en C avec la librairie SDL. Le joueur compose et gère une équipe de “Pocket Monsters” afin de relever divers défis.

Fonctionnalités Principales

  • Interface graphique gérée via la SDL
  • Système de combat au tour par tour
  • Gestion et personnalisation d’équipes
  • Possibilité de soigner et de changer de créatures
  • Sauvegarde et chargement de partie en cours d’implémentation

Implémentation et Objectifs

Le projet illustre l’architecture modulaire d’un petit moteur de jeu. Divers commits GitHub démontrent l’évolution du code : corrections de bugs, ajouts de fonctionnalités, et mise en place de tests unitaires pour améliorer la fiabilité de l’application.
