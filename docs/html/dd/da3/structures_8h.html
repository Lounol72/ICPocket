Description détaillée

Définition des structures de données utilisées dans le jeu.

Auteur
William Nathan

Ce fichier contient les définitions des structures de données utilisées pour représenter les ICmon, les attaques, les équipes et d'autres éléments du jeu. Il inclut également des énumérations pour les types de ICmon, les catégories d'attaques et les effets d'état.
