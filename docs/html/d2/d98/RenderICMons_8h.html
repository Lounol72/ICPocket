Description détaillée

Définition et gestion du rendu des ICMons.

Ce fichier définit la structure IMG_ICMons et les fonctions associées pour le rendu des sprites ICMons dans une application SDL.

Initialise un sprite ICMons.

Cette fonction crée et initialise le sprite associé à un poke, incluant le chargement de l'image, le retournement horizontal si nécessaire, la création des textures, ainsi que la configuration des textes et barres de points de vie.

Paramètres
rendererLe renderer SDL.
spriteRectLe rectangle du sprite.
nameRectLe rectangle pour le nom.
pvRectLe rectangle pour la barre de points de vie.
pokeLe pointeur vers la structure t_Poke contenant les informations du poke.
fontLa police TTF utilisée pour le texte.
teamL'équipe (si 1, l'image est retournée horizontalement).
Renvoie
IMG_ICMons* Le pointeur vers le sprite initialisé, ou NULL en cas d'erreur.
