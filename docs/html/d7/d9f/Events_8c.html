Initialise les sprites pour tous les pokémons d'une équipe.

Initialise les sprites des équipes.

Cette fonction initialise le sprite de chaque pokémon dans une équipe en définissant les rectangles de positionnement pour le sprite, le nom et la barre de points de vie.

Note
Le type Team doit être défini dans vos en-têtes (ex. via un typedef avec les champs nb_poke et team).
Paramètres
winPointeur sur la fenêtre.
teamSpritePointeur sur la structure représentant l'équipe.
x_ratioRatio de positionnement horizontal du sprite.
y_ratioRatio de positionnement vertical du sprite.
teamFlagIndicateur d'équipe (0 pour l'équipe rouge, 1 pour l'équipe bleue).
Renvoie
int Retourne 0 en cas de succès, -1 en cas d'erreur.

Cette fonction configure les sprites des équipes en fonction des ratios et des drapeaux spécifiés.

Paramètres
winPointeur vers la structure Window contenant l'état du jeu.
teamSpritePointeur vers la structure des sprites de l'équipe.
x_ratioRatio X pour le positionnement du sprite.
y_ratioRatio Y pour le positionnement du sprite.
teamFlagDrapeau de l'équipe : 0 = rouge, 1 = bleu.
Renvoie
Retourne 0 en cas de succès, une valeur différente en cas d'erreur.
