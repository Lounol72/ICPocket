Questions fréquentes

Comment puis-je commencer à contribuer?

Pour commencer à contribuer, nous vous recommandons de parcourir les tickets ouverts sur GitHub marqués "good first issue". Vous pouvez aussi consulter notre guide de contribution dans le fichier CONTRIBUTING.md sur le répertoire GitHub.

Dois-je être développeur pour contribuer?

Non! Nous accueillons toutes sortes de contributions : tests, design, documentation, traduction, idées de gameplay, et bien sûr, du code si vous avez les compétences.

Comment sont évaluées les contributions?

Toutes les contributions sont examinées par notre équipe de développement. Nous cherchons des ajouts qui améliorent l'expérience de jeu, maintiennent la stabilité et respectent l'orientation du projet.
