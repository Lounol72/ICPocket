ICPocket

À propos du jeu

ICPocket est un jeu "Pokemon-like" développé par des étudiants passionnés de l'Université du Mans. Plongez dans un univers unique où la programmation rencontre l'aventure !

Configuration requise

  • OS: Windows 10/11 ou Linux
  • Processeur: Dual Core
  • Mémoire: 2 GB RAM
  • Graphiques: Compatible OpenGL
  • Stockage: 500 MB
